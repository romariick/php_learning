1. Notes :

a. 
- class : c'est une réprenstation d'un objet ou un modèle de donnée 
- objet : c'est une instance d'un class
- Une classe contient des attributs et des méthodes.

b. Surété de la programmation orienté Object

- public : tous les autres classes à le droits d'accès ou d'utilser le méthode 
ou attribut
- private : utilisé seulement à l'intérieur de la class elle même
- protected : utilisé par la classe elle même et la classe qui hérite
