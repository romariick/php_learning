A. Définition :

- class : c'est une répresentation d'un objet ou un modèle de donnée ;

- objet : c'est une instance d'un class ;

- Une classe contient des attributs et des méthodes ;


B. Surété de la programmation orienté Object

- public : tous les autres classes à le droits d'accès ou d'utilser le méthode
ou attribut ;

- private : utiliser seulement à l'intérieur de la class elle même ;

- protected : utiliser par la classe elle même et la classe qui hérite.
